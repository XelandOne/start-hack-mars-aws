# Product Overview

This is a hackathon project: a web application for managing a **tomato plantation on Mars**.

The app helps users monitor and manage crop data including soil conditions, growth cycles, and water requirements for Martian agriculture.

A Mars Crop Knowledge Base is available via MCP server to provide realistic crop data (soil conditions, growth stages, water needs, etc.). Use it to inform features and UI.

The project is built during a time-boxed hackathon — favor working software over perfection. Start simple (basic CRUD), then layer on features.
