## Key Data Points

- Crew: 4 astronauts, 450-day mission
- Daily caloric need: ~12,000 kcal/day total
- Daily protein need: ~360–540 g/day
- Recommended area allocation: 40–50% potatoes, 20–30% legumes, 15–20% leafy greens, 5–10% radishes/herbs
- Optimal CO₂ range: 800–1200 ppm
- Optimal temperature (most crops): 15–25°C
- Optimal humidity: 50–70% RH
- Water recycling efficiency target: >85–95%
